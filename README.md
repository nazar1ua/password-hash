# password-hash

Simple Go program for generating password hashes

## Installing

1. Clone (or download) this repo
2. Build this for your machine (only linux or mac)
  ```shell
  make build
  ```
  Also you can run
  ```shell
  go build -o <path/to/user/bin>
  ```
  for Windows

## Usage

```shell
password-hash -p <password> -c <cost>
```
